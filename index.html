Cloud Manager 설명서

Cloud Manager provides IT experts and cloud architects with a centralized control plane to manage, monitor, and automate data in hybrid cloud environments. Browse the docs by product area or use the search above to find content anywhere in the Cloud Manager docs.

설정 및 관리

Cloud Manager 사용을 시작하고 커넥터, 계정 등을 관리하는 방법에 대해 알아보십시오.

릴리스 정보

모든 Cloud Manager의 새로운 기능에 대해 알아보십시오.

Active IQ 디지털 자문업체

최적의 스토리지 상태를 위해 실행 가능한 인텔리전스

ONTAP용 Amazon FSx

ONTAP의 모든 기능을 AWS로 관리하는 네이티브 서비스로 사용하십시오.

API를 참조하십시오

클라우드 기반 리소스를 프로그래밍 방식으로 관리합니다.

AppTemplate(애플리케이션 템플릿)

Cloud Manager에서 작업을 자동화하여 프로비저닝 속도를 높이고 수동 작업을 제거합니다.

Azure NetApp Files

핵심 비즈니스 애플리케이션을 위한 고성능 파일 스토리지 서비스입니다.

클라우드 백업

클라우드 및 온프레미스 데이터 백업 및 복원

클라우드 데이터 감지

애플리케이션 데이터 및 클라우드 환경의 개인 정보를 보호합니다.

Cloud Sync

온프레미스 및 클라우드 데이터 저장소 간에 데이터를 동기화합니다.

클라우드 계층화

사내 ONTAP 스토리지를 클라우드로 확장

Cloud Volumes ONTAP

클라우드 스토리지를 보다 경제적이고, 지능적이며, 규정을 준수하고, 안전하게 보호하십시오.

AWS 환경을 위한 Cloud Volumes Service

All-Flash 성능을 지원하는 클라우드 네이티브 파일 서비스

Google Cloud용 Cloud Volumes Service

파일 스토리지를 보호하고 속도를 높이십시오.

컴퓨팅

클라우드 컴퓨팅 지출에 대한 가시성을 확보하고 잠재적 절감을 파악하십시오.

글로벌 파일 캐시

분산된 인력을 위해 글로벌 파일을 실시간으로 공유할 수 있습니다.

Kubernetes 클러스터

모니터링

온프레미스 ONTAP 클러스터

하이브리드 클라우드 전반에서 NetApp AFF 및 FAS 시스템을 관리할 수 있습니다.
